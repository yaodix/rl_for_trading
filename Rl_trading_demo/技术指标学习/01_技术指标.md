### 📊 五大类指标信号详解

| 指标类别                          | 常用指标与参数                                                     | **核心交易信号** (买/卖/警示)                                                                                                                                                                                              | 信号逻辑与重要补充                                                                                                                                                    |
| :-------------------------------- | :----------------------------------------------------------------- | :------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | :-------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **1. 趋势指标** (滞后)      | **50日EMA**、**200日EMA**                              | **“金叉” (买入)**：50日EMA上穿200日EMA。`<br>`**“死叉” (卖出)**：50日EMA下穿200日EMA。                                                                                                                         | **代表趋势转变**。短期均线穿越长期均线，被视为趋势反转的确认信号。                                                                                              |
| **2. 均值回归指标** (滞后)  | **布林带 (20, 2)**`<br>`(中轨:20日SMA, 上下轨:±2倍标准差) | **潜在买入**：价格触及或跌破**下轨**。`<br>`**潜在卖出**：价格触及或突破**上轨**。`<br>`**爆发警示**：**带宽极度收缩**（挤压状态）。                                                     | 代表**超买/超卖**。价格触碰轨道不直接等于反转信号，需结合其他指标确认。`<br>`带宽收缩后常接**大幅波动**。                                               |
| **3. 相对强弱指标** (领先)  | **随机指标 (14,7,3)**                                        | **买入信号**：`<br>`1. 在**超卖区 (20以下)** 形成金叉。`<br>`2. 两线从超卖区**回升**。`<br>`**卖出信号**：`<br>`1. 在**超买区 (80以上)** 形成死叉。`<br>`2. 两线从超买区**回落**。 | 代表**买卖力循环**。**核心原则**：超买/超卖区的交叉比普通交叉**更可靠**。在强趋势中，指标可能**钝化**（长期在超买区），此时死叉可能是假信号。 |
| **4. 动量指标** (领先)      | **MACD (12, 26, 9)**                                         | **买入信号**：**快线(DIF)上穿慢线(DEA)**，形成金叉，且柱状图由负转正。`<br>`**卖出信号**：**快线下穿慢线**，形成死叉，且柱状图由正转负。`<br>`**趋势减弱**：**柱状图与价格背离**。       | 代表**价格变化速度**。**背离**是重要警示：价格创新高，但MACD柱状图高点降低 (**顶背离**)，预示下跌；反之是**底背离**，预示上涨。               |
| **5. 量能指标** (领先/滞后) | **平衡交易量 (OBV)**                                         | **趋势确认**：**OBV与价格同步**创新高/低。`<br>`**背离预警**：**OBV与价格走势相反**。`<br>`**突破信号**：OBV的**趋势线被突破**。                                                         | 代表**资金流向**。OBV通过累计成交量衡量买卖压力。**背离是强烈预警**：如价格创新高，但OBV未创新高，表明上涨缺乏成交量支持。                                |

### 💎 总结：初学者的核心信号清单

对于新手，不需要同时使用所有指标。建议从每类中选一个，组合成你的“仪表盘”：

1. **看趋势方向** → **200日EMA**。价格在线上方为多头市场，反之为空头。
2. **看相对强弱** → **随机指标**。观察是否处于超买/超卖区。
3. **看动量变化** → **MACD**。观察快慢线金叉/死叉及柱状图变化。
4. **看资金验证** → **OBV**。观察与价格的背离。
